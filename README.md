# Recetas Familia — React + Vite + CSS (Español)

Mini web para ver el menú de 2 semanas (desayuno, comida con receta completa, lunch niña sin recalentar, lunch adulto con recalentado y cena ligera).

## Requisitos
- Node.js 18+ recomendado

## Instalar y correr
```bash
npm install
npm run dev
```

Abre el enlace que te sale en la terminal (normalmente http://localhost:5173).

## Compartir con tu esposa
- Opción 1: correr en tu compu y abrir en el celular dentro de la misma red (más avanzado).
- Opción 2 (recomendado): subir a Netlify/Vercel (muy fácil con Vite).

## Contenido editable
- `src/data/plan.js` contiene TODAS las recetas (2 semanas) en español.
